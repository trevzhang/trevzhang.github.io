Optionalзҡ„orElse()дёҺorElseGet()

еҸ‘иЎЁдәҺ 2020-12-30 еҲҶзұ»дәҺ Java

йЎ№зӣ®дёӯжңүиҝҷж ·дёҖж®өд»Јз Ғпјҡ

еҠҹиғҪжҳҫиҖҢжҳ“и§ҒпјҢservice.A()еҰӮжһңиҝ”еӣһеҖјжҳҜnullпјҢеҲҷиҝ”еӣһservice.B()пјҢеҗҰеҲҷзӣҙжҺҘиҝ”еӣһservice.A()гҖӮ
е®һйҷ…дҪҝз”ЁдёӯеҸ‘зҺ°пјҡ
еҰӮжһңservice.A()иҝ”еӣһйқһnull,жңҖз»Ҳз»“жһңжҳҜservice.A(),з„¶иҖҢservice.B()иҝҷдёӘж–№жі•д№ҹиў«жү§иЎҢдәҶгҖӮиҝҷж ·иӮҜе®ҡе°ұдёҚеҜ№дәҶ,еҰӮжһңservice.B()дёӯиҝҳжңүжҸ’е…Ҙж•°жҚ®еә“жҲ–иҖ…RPCиҝҷз§Қж“ҚдҪңпјҢй—®йўҳе°ұеӨ§дәҶгҖӮеҲҡејҖе§Ӣиҝҳд»ҘдёәжҳҜд»Җд№Ҳжү§иЎҢйЎәеәҸй—®йўҳпјҢеҗҺжқҘеңЁStack OverflowдёҠзңӢеҲ°иҖҒеӨ–и®Ёи®әorElse()е’ҢorElseGet()зҡ„еҢәеҲ«пјҢе…¶дёӯдёҖзӮ№еҢәеҲ«е°ұжҳҜпјҡ

orElse(T)ж— и®әеүҚйқўOptionalе®№еҷЁжҳҜnullиҝҳжҳҜnon-nullпјҢйғҪдјҡжү§иЎҢorElseйҮҢзҡ„ж–№жі•пјҢorElseGet(Supplier)е№¶дёҚдјҡпјҢеҰӮжһңserviceж— ејӮеёёжҠӣеҮәзҡ„жғ…еҶөдёӢпјҢOptionalдҪҝз”ЁorElseжҲ–иҖ…orElseGetзҡ„иҝ”еӣһз»“жһңйғҪжҳҜдёҖж ·зҡ„гҖӮ

stack overflowдёҠжңүдәәиҝҳз»ҷеҮәиҝҷж ·дёҖдёӘдҫӢеӯҗпјҡ

вҶ‘дёҠйқўжҸҗй—®иҖ…еӨ§жҰӮзҡ„ж„ҸжҖқе°ұжҳҜпјҢorElse()е’ҢorElseGet()йғҪиғҪеңЁOptionalе®№еҷЁзҡ„еҖјдёәз©әж—¶жҸҗдҫӣдёҖдёӘеҸҜйҖүзҡ„еҖјпјҢз”ЁorElse(new Object())еҸҜд»ҘзӣҙжҺҘдј е…ҘеҜ№иұЎпјҢиҖҢз”ЁorElseGet(Supplier)ж—¶йңҖиҰҒдј е…ҘдёҖдёӘSupplierпјҲеҸҜд»ҘдёәLamdaиЎЁиҫҫејҸпјүпјҢй—®иҝҳжңүжІЎжңүе…¶д»–еҢәеҲ«гҖӮ

вҶ‘дҪңиҖ…йҮҮз”Ёзҡ„жңҖдҪізӯ”жЎҲжҳҜиҝҷж ·зҡ„

зңӢдәҶдёҠйқўд»Јз ҒпјҢжҲ‘е°ұжҠҠжҲ‘зҡ„д»Јз Ғж”№жҲҗеҰӮдёӢеҚіеҸҜпјҡ

з»“и®әпјҡ

иӢҘж–№жі•дёҚжҳҜзәҜи®Ўз®—еһӢзҡ„пјҢдҪҝз”ЁOptionalзҡ„orElse(T)пјӣ

иӢҘжңүдёҺж•°жҚ®еә“дәӨдә’жҲ–иҖ…иҝңзЁӢи°ғз”Ёзҡ„пјҢйғҪеә”иҜҘдҪҝз”ЁorElseGet(Supplier)гҖӮ
